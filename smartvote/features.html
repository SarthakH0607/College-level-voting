⭐ Platform Features

Everything you need for secure college elections

🗳️

Position-Based Voting

Vote for CR, Vice CR, Treasurer, and more — each position handled separately for fair representation.

📷

Face Recognition

Teachers verify identity through InsightFace-powered facial recognition before casting their vote.

🎙️

Voice Verification

Dual biometric security with Canary-Qwen 2.5B voice recognition and passphrase matching for teacher elections.

📊

Real-Time Results

Watch vote counts update live with animated progress bars powered by Firestore real-time snapshots.

🛡️

Duplicate Prevention

One vote per user per election, enforced through Firestore transactions and security rules.

📱

Fully Responsive

Works beautifully on desktop, tablet, and mobile devices with adaptive layouts.

⚙️

Admin Panel

Complete election management — create elections, add candidates, manage users, and reset biometrics.

🎨

Role-Based Theming

Students see blue, teachers see purple, admins see slate — instant visual context for every user.
